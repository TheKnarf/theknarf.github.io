# Productivity Apps

Here is a list of some Productivity apps that I've found that looks interesting.

None of these I've properly tested.

## Reference managers & Capture tools

- [Kosmik](https://www.kosmik.app/) - A browser and canvas solution all in one

- [Artifacts](https://x.com/zachwaugh/status/1794042990236684315) - A nice looking UI for organizing links and images

- [Eagle](https://en.eagle.cool/) - Reference manager used by professionals

- [Lazy](https://x.com/demtzu/status/1589633050107080704) - Raycast-looking UI for quick capture anywhere

- [Archivist](https://github.com/szymonkaliski/archivist) - Tool for archiving and exploring

## Reading / RSS

- [Matter](https://hq.getmatter.com/) - Reader app

## Bookmark managers & archiving

- [linkding](https://github.com/sissbruecker/linkding) - selfhosted bookmark manager

- [kiwix](https://kiwix.org/en/) - Offline access

- [MarkDdownload](https://github.com/deathau/markdownload) - Markdown web clipping

- [Linkwarden](https://github.com/linkwarden/linkwarden) - Bookmark preservation for individuals and teams

- [Grimoire](https://github.com/goniszewski/grimoire) - Bookmark manager for the wizards

- [Raindrop.io](https://github.com/raindropio) - Bookmark manager

- [Archivebox](https://archivebox.io/) - Open source self-hosted web archiving

- [Paperless-ngx](https://docs.paperless-ngx.com/) - Open-source document management system

- [Perkeep](https://perkeep.org/) - permanently keep your stuff

## Todo-lists & calender apps

- [benjiDotSo](https://x.com/thekitze/status/1654920486961262598) - Nice looking calender / pomodoro timer

- [Tiblo](https://x.com/zehf/status/1651201514461904896) - Nice looking tasklist and calender timeblocking

- [Sunsama](https://www.sunsama.com/) - Todos & calender timeblocking

## Others

- [Pile](https://github.com/UdaraJay/Pile/tree/v0.7.15) - Cute journaling app

- [Sukha](https://www.thesukha.co/) - Body doubeling app

- [recall](https://www.getrecall.ai/) - AI system to recall stuff you've seen
