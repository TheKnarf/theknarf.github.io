# Signed Distance Fields (SDFs)

Signed Distance Fields is a way to represent 3D models as implicit surfaces.
to use them in video games you may want to turn them into a mesh.

## Meshing algoritms

Algoritm        | How to handle LODs   | Libraries
----------------|----------------------|-----------
Marching cubes  |[Transvoxel](https://transvoxel.org/?utm_source=chatgpt.com) |
Surface Nets    |Skirting / stitching  | [fast-surface-nets-rs](https://github.com/bonsairobo/fast-surface-nets-rs), [GDvoxelTerrain](https://github.com/JorisAR/GDVoxelTerrain/blob/main/src/voxel_terrain/meshing/stitched_surface_nets/stitched_mesh_chunk.cpp)
Dual Contouring |Can work on octrees   |
